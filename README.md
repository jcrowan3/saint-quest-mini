# Saint Quest

A Catholic saints trivia and adventure game. Choose a saint, journey through their story, answer challenges, and build your virtue profile.

## Play Now

**https://jcrowan3.github.io/saint-quest-mini/**

Just open the link — no install needed. Works on desktop and mobile.

## Features

- 8 saints with unique stories and 80+ quests
- Dilemma and trivia challenges
- 3 difficulty levels (Easy / Normal / Hard with timer)
- Streak & combo system with bonus virtues
- Bonus round with cross-saint trivia
- Achievement system (8 achievements with toast notifications)
- Daily Challenge mode with shareable results
- High score tracking per saint
- Radar chart virtue profile
- Particle effects, sound, and animations

## Running Locally

Open `docs/index.html` in any browser — it's a single self-contained file with no dependencies.

## Legacy Streamlit Prototype

The original MVP prototype (`app.py`) used Streamlit with 2 saints and basic gameplay. It has been superseded by the HTML version above.

```bash
pip install -r requirements.txt
streamlit run app.py
```
